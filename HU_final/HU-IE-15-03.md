# HU-IE-15-03: Creación Automática de Proveedores

**Proceso:** IMPORTACIÓN DE EQUIPOS CAT  
**Subproceso:** GESTIÓN DE PROVEEDORES  
**Requerimiento Original:** REQ-15 — Los proveedores deben crearse automáticamente en el SII a partir de la información de D365 usando el código de 6 dígitos.  
**Versión:** 2.0  
**Fecha de revisión:** 2026-03-05  

---

## Descripción
Si durante la sincronización de facturas el sistema detecta un código de proveedor (6 dígitos) que no existe en la base de datos del SII, debe crearlo automáticamente con datos básicos para no detener el proceso de carga.

### Reglas:
- Usar exclusivamente el código de 6 dígitos de D365.
- Sincronización unidireccional (D365 -> SII).
- El usuario podrá completar datos adicionales (NIT, Dirección) posteriormente.
