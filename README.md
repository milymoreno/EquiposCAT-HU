# EquiposCAT – Historias de Usuario de Importación de Equipos CAT

Repositorio de historias de usuario (HU), documentación de ajustes y glosarios para el proceso de **Importación de Equipos CAT**.

## Estructura
- `HU_Inicial/` – HUs base del levantamiento inicial
- `HU_final/` – HUs enriquecidas y revisadas con sesiones
- `DOCUMENTACION_AJUSTES/` – Diferencias justificadas y glosarios por HU
- `REQUERIMIENTOS/` – Requerimientos originales del proceso
- `TRANSCRIPT/` – Transcripciones de sesiones de levantamiento
- `MODELO_DATOS/` – Modelos de datos asociados al proceso
