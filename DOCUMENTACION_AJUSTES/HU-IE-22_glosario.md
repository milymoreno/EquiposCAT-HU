# HU-IE-22 — Glosario de Términos

Términos técnicos y administrativos relacionados con el cierre del proceso de importación.

| Término | Definición |
| :--- | :--- |
| **Atomicidad** | Propiedad de una transacción en la que todas sus acciones se realizan o ninguna, evitando estados parciales. |
| **Consolidado Definitivo** | Estado final de un embarque donde la información ya fue validada y transmitida exitosamente al ERP. |
| **Idempotencia** | Capacidad del sistema para procesar la misma petición múltiples veces sin cambiar el resultado inicial (evita duplicados). |
| **Landed Cost** | Costo total acumulado de un equipo una vez ha llegado al almacén (FOB + fletes + seguros + aranceles + gastos locales). |
| **OData / REST** | Protocolos de comunicación utilizados para la integración de datos entre el SII y Dynamics 365. |
| **SIPDYNIF** | Prefijo de las tablas técnicas intermedias donde se prepara la información para el envío a las interfaces del ERP. |
