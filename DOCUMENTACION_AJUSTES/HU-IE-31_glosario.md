# HU-IE-18-01 — Glosario de Términos: Parcialización

Términos clave para el proceso de división de documentos de transporte.

| Término | Definición |
| :--- | :--- |
| **Parcializar** | Acción de dividir un documento de transporte original en uno o más documentos adicionales, manteniendo la trazabilidad con el origen. |
| **HBL Parcial** | Nuevo documento de transporte generado a partir de una parcialización, con su propio identificador único en el SII. |
| **División por Unidades** | Capacidad de separar una cantidad específica de una misma referencia (Ej: de un registro de 10 motores, mover 3 a otra guía). |
| **Saldo Pendiente** | Cantidad de unidades o referencias que permanecen en el documento original después de una parcialización. |
