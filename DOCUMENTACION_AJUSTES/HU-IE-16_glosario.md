# HU-IE-16 — Glosario de Términos de Negocio

**Aplica a:** HU-IE-16 – Integración con Logística Internacional para Recepción de BL  
**Fecha:** 2026-03-05  

---

- **BL (Bill of Lading / Conocimiento de Embarque):** Es el "título de propiedad" de la mercancía durante el transporte internacional. En el SII funciona como el documento principal que agrupa las facturas de un envío.
- **Dealer (Distribuidor/Destino):** Código técnico que identifica el destino final de la carga o el puerto de llegada. Ej: El código **R490** se usa para identificar que la carga entra por el Puerto de Buenaventura.
- **E / R (Equipos / Repuestos):** Prefijos usados en los códigos para diferenciar qué tipo de mercancía se está moviendo. Es vital para que el sistema aplique las reglas de validación correctas.
- **Guía:** Sinónimo de BL en el contexto de transporte. Es el número identificador del viaje de la mercancía.
- **Puerto de Llegada:** Lugar físico (puerto marítimo o terminal terrestre) donde se recibe la importación. Se asigna automáticamente según el código de Dealer recibido.
