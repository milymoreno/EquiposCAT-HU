# HU-IE-15-04 — Glosario de Términos

Términos relacionados con la gestión de excepciones y errores de integración.

| Término | Definición |
| :--- | :--- |
| **Log de Errores** | Registro histórico y detallado de los fallos ocurridos durante una comunicación entre sistemas o procesamiento de archivos. |
| **Re-sincronización** | Proceso de volver a solicitar la información de una factura al ERP D365 después de que se han corregido los errores de origen. |
| **Umbral de Error** | Porcentaje máximo permitido de fallos en una carga masiva antes de disparar una alerta crítica al responsable. |
| **Acción Correctiva** | Modificación manual o automática realizada para subsanar un error de validación (Ej: asignar un Incoterm faltante). |
